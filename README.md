CampusSchedule
===

"CampusSchedule" is an Android application for schedule management mainly for university students.

## Description

Usually for university students, but any students else and the others can use this application.
At the time you would like to check school schedule, you can view a brief outline as timetable or list, 
and also see the details such as course names, teachers, textbooks, grade points, and etc.
Additionally, you can manage textbooks as list and search ones with Google books.

## Feature
<dl>
  <dt>Table and List</dt>
  <dd>
    Most of applications for university students have only table view. 
    However, this "CampusSchedule" supports both forms of table and list.Acoo 
    Table helps to grasp schedule through a week. 
    On the other hand, list assists to check details
  </dd>
  
  <dt>Dynamic and flexible view</dt>
  <dd>
    Life style of students is very various.
    For example, do you have classes on Saturdays? 
    How many classes do you have in a day?
    You can adjust view (especially table one) according to the answers.
  </dd>
  
  <dt>Colorful days</dt>
  <dd>
    Grid colors of table is classified by course types. 
    You can customize the colors in settings.
  </dd>
  
  <dt>Textbooks management</dt>
  <dd>
    One of bothersome things for students is how to manage textbooks.
    In "CampusSchedule", you can easily deal it by some functions. 
    One is to manage together other information of the course such as the name and type.
    Another function is special list for books.
    You can see what to purchase and bring.
    In addition, book search function is available.
    If you don't know complete book information, you can search on web.
  </dd>
</dl>
